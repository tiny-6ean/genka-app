# 原価計算ミニアプリ｜静灯工房

ハンドメイド制作のための  
**静かでシンプルな原価計算ツール**です。

材料費・制作時間・利益を  
落ち着いて確認できるように設計しています。

---

## 機能
- 材料費の入力  
- 制作時間の入力  
- 時給換算  
- 利益計算  
- 1ページで完結するミニマル設計

---

## コンセプト
静灯工房の世界観である  
**淡色・静けさ・余白**を大切にした  
ミニマルな計算ツールです。
