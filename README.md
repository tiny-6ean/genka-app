# 原価計算ミニアプリ｜Tiny Bean Studio

このミニアプリは、ハンドメイド制作のための  
**シンプルな原価計算ツール**です。

材料費・制作時間・販売価格のバランスを  
静かに、落ち着いて確認できるように設計しています。

---

## 🌿 機能
- 材料費の入力  
- 制作時間の入力  
- 時給換算  
- 利益計算  
- シンプルな1ページ構成

---

## 🌱 コンセプト
Tiny Bean Studio（静灯工房）の世界観である  
**淡色・静けさ・余白**を大切にした  
ミニマルな計算ツールです。
